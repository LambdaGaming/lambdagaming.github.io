Lambda Gaming Sandbox

If you are a returning member, welcome back! If you are new here, welcome to the server! The rules for the server are on the left, and info about this server is on the right. You can access this information and more after you have spawned in by typing !rules in chat.



Server Rules:

  1. Don't build in the middle of high traffic areas. Examples of high traffic areas: Roads, footpaths, train tracks, spawn, any other areas that are regularly accessed by players and aren't regularly used for bases.

  2. Don't abuse buildmode. Example of abuse: noclipping to get into someone's base and killing them, or walking into an enemy base with god mode on to see the layout of the base without risk of dying.

  3. Don't damage WAC Aircraft/Star Wars vehicles until they are off of the ground and flying.


About This Server:

As long as you abide by the server and community rules, you can basically do whatever you want here. We run a large variety of addons from our other servers and player suggestions.

Notes:

  • This server does not run 24/7. If you want to be notified the next time it opens, join the Discord. The link can be found on our Steam group page, which can be accessed by pressing !group in chat.

  • This server does not use FastDL due to host limitations. If you need the addons, type !addons in chat once you have loaded in.
