Welcome and enjoy your stay! The rules for the server are on the left, and info about this server is on the right. You can view the community rules and more after you have spawned in by typing !rules in chat.



Gamemode-Specific Rules:

These are rules for specific gamemodes that will only be enforced if broken on the gamemode the rule is written for.


Cops and Runners:

  1. Try to avoid camping player spawns as cop.

Sledbuild:

  1. Every player is given basic admin privileges to allow for more creative building. Don't abuse these privileges.

TTT:

  1. Only kill players as innocent if you have reason to believe they are a traitor.



About This Server:

As the name implies, this server hosts a variety of gamemodes, most of which are round-based. We usually stick to one gamemode until a few players start requesting a different one, then we hold a vote.

List of Current Gamemodes:

Cops and Runners, Darkest Days, Dropzone, Flood, Homicide, Jazztronauts, Prop Hunt, Pirate Ship Wars Remix, Sledbuild, Stop it, Slender!, TTT, Trash Compactor, and Zombie Survival.


Notes:

  • This server does not run 24/7. If you want to be notified the next time it opens, join the Discord. The link can be found by typing !discord in chat.

  • If you are missing the addons, you may need to enable multiplayer downloads in the game options. If you prefer to manually download them, you can type !addons in chat once you have loaded in.
