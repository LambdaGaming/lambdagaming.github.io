About This Server:

As the name implies, this server hosts a variety of gamemodes, most of which are round-based. Gamemodes and maps are changed based on player votes.

List of Current Gamemodes:

  • Cops and Runners
  • Darkest Days
  • Dropzone
  • Flood
  • Jazztronauts
  • MelonRacer
  • Prop Hunt Enhanced Plus
  • Pirate Ship Wars Remix
  • Sledbuild
  • Stop it, Slender!
  • TTT2
  • Trash Compactor
  • Zombie Survival Classic

Notes:

  • This server does not run 24/7. If you want to be notified the next time it opens, join the Discord. You can get the link by typing !discord in chat.

  • If you are missing the addons, you might need to enable multiplayer downloads in the game options. If you prefer to manually download them, type !addons in chat.

  • To see the rules and other resources, type !rules in chat.

  • Type !rtv in chat to rock the vote for changing maps and/or gamemodes.
