Gamemode-Specific Rules:

These are rules for specific gamemodes that will only be enforced if broken on the gamemode the rule is written for.


Cops and Runners:

  1. Try to avoid camping player spawns as cop.

Sledbuild:

  1. Every player is given basic admin privileges to allow for more creative building. Don't abuse these privileges.

TTT:

  1. Only kill players as innocent if you have reason to believe they are a traitor.





About This Server:

As the name implies, this server hosts a variety of gamemodes, most of which are round-based. We usually stick to one gamemode until a few players start requesting a different one, then we hold a vote.

List of Current Gamemodes:

Cops and Runners, Dropzone, Flood, Homicide, Jazztronauts, Prop Hunt, Pirate Ship Wars Remix, Pirates Vs. Boners, Sledbuild, Stop it, Slender!, TTT, Trash Compactor, and Zombie Survival.


Notes:

  • This server does not run 24/7. If you want to be notified the next time it opens, join the Discord. The link can be found on our Steam group page, which can be accessed by pressing !group in chat.

  • This server does not use FastDL due to host limitations. If you need the addons, type !addons in chat once you have loaded in.

  • The rules on the left side of this loading screen only cover rules for certain gamemodes. For community-wide rules, type !rules in chat.
