Minecraft Server Rules

Note: The following rules apply to all minecraft servers except Unbalanced.

Note: Overhead images of the maps on some of the servers can be found here.


  1. Along with these rules, be sure to read the community-wide rules which can be found by clicking here.

  2. PVP is enabled, you can choose to base with someone or by yourself, who your allies and enemies are is up to you.

  3. You can only use reasonable force to raid a base, using TNT to blow up a small wooden hut, or burning a mansion just to loot a chest isn't allowed.

  4. You don't have to announce a raid, but you can only raid/loot bases that are occupied, unless they have officially been declared abandoned by the owner.

  5. Using game exploits of any kind is not allowed. This includes using spawning algorithms to your advantage.
