Minecraft Server Rules

Note: Along with these rules, be sure to read the community-wide rules which can be found here. Overhead images of the server maps can be found here.


General Rules:

These rules apply to all Minecraft servers except the Unbalanced servers.

  1. PVP is enabled. You can choose to base with someone or by yourself, who your allies and enemies are is up to you.

  2. You can only use reasonable force to raid a base. For example, using TNT to blow up a small wooden hut or burning a mansion just to loot a chest isn't allowed, but using TNT to blow a hole in the side of a mansion to gain access to a chest is allowed. Ask an admin if you are unsure whether or not your raid method is reasonable.

  3. You don't have to announce a raid, but you can only raid/loot bases that are occupied, unless they have been declared abandoned by the owner.

  4. Using game exploits of any kind is not allowed.

  5. Griefing is not allowed. Refer to the raiding rules above for what you're allowed to do if you want to mess with someone.

Unbalanced Rules:

These rules only apply to the original Unbalanced server.

  1. Causing damage to a protected structure or causing damage to a player who is inside a protected structure is not allowed.

  2. Players can declare something as a protected structure as long as it is owned by them and the structure has no weapons in or around it.

Unbalanced 2 Rules:

These rules only apply to the Unbalanced 2 server.

  1. Causing damage to a protected structure or causing damage to a player who is inside a protected structure is not allowed.

  2. Players can declare something as a protected structure as long as it is owned by them and the structure has no weapons in or around it.

  3. Using the red matter bomb without superadmin permission is not allowed.

  4. Using chance cubes near active bases is not allowed.

  5. Structures owned by a disconnected player are to be considered protected structures.
