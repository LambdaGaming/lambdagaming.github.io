CityRP Chat Commands

The following is a list of non-DarkRP chat commands that players can use on CityRP. If you are viewing this from in-game, a list of chat commands for the DarkRP gamemode specifically can be found on the left.

Gameplay Commands

!alert

Alerts all police and firefighters to your location. Requires that you have life alert, which can be purchased at the NPC shop.

!alertcheck

Tells you whether or not you have life alert equipped.

!viewproperties

Outlines every ownable property with a wireframe box so you can see where the boundaries are. Type this command again to remove the outlines.

!firemayor

Casts a vote to fire the mayor. At least 75% of all players on the server must vote for it to pass. Intended to be used if the mayor is doing a terrible job and the citizens want him removed from office peacefully. If a vote successfully passes, the mayor is instantly fired and he cannot enter another election for 30 minutes. A vote will fail after 5 minutes if not enough people vote.

URL Commands

These commands will open URLs for various community and server-related sites. You will be prompted before the URLs open in your Steam overlay.

!addons

Opens the page to the server addon collection.

!discord

Prints the link to the Lambda Gaming Discord in chat.

!group

Opens the page to the Lambda Gaming Steam group.

!rules

Opens the CityRP Rules and Guidelines page. This page is also accessible from the F1 menu if you are using the x86-64 beta branch.
