Chat Commands

The following is a list of chat commands and their functions for the CityRP server. Angled brackets { } in a command means it's a required argument. Square brackets [ ] in a command means it's an optional argument. If you are viewing this from in-game, a list of chat commands for the DarkRP gamemode specifically can be found on the left.

URL Commands

These commands will open URLs for various community and server-related sites. You will be prompted before the URLs open in your Steam overlay.

!addons

Opens the page to the server addon collection.

!group

Opens the page to the Lambda Gaming Steam group.

!rules

Opens the CityRP Rules and Guidelines page. This page is also accessible from the F1 menu if you are using the x86-64 beta branch.

Gameplay Commands

These commands are intended to be used during normal gameplay.

!alert

Alerts all police and firefighters to your location. Requires that you have life alert, which can be purchased at the NPC shop.

!alertcheck

Tells you whether or not you have life alert equipped.

!showid

Prints in chat your real name, fake name, occupation, and government relationship.

!currentevent

Prints the current event in chat. Prints N/A if there is currently no event active.

!placehit

Opens the hitman menu where players can place a hit on someone they don't like. Note that placing a hit costs a certain amount of money depending on what you enter.

!undercover

Changes your job to Undercover Officer without notifying the rest of the server. It also changes your job name to Citizen to prevent your cover from being blown on the scoreboard.

Mayor Commands

Only the mayor can use these commands.

!fixeco

Adds 30 eco points at the cost of $300 from the mayor funds. Will not work if there is less than $300 in the mayor funds.

IA Agent Commands

Only the IA Agent can use these commands.

!firemayor

Starts a vote to fire the mayor. 75% of players must vote in favor for it to pass. Intended to be used if the mayor is doing a terrible job and the citizens want him removed from office peacefully.

Bus Driver Commands

Only bus drivers can use these commands.

!busstop

Used during the bus passenger random event when the bus driver is in range of the destination to let the passenger know they can get off.

Admin Commands

Only server admins can use these commands.

!fireoff

Admin only. Extinguishes all of the fires currently burning on the map.

!noclip [target]

Admin only. Allows players to fly through solid objects. Leaving the target argument blank will enable noclip on the calling player.

!god [target]

Admin only. Prevents the target player from being damaged. Leaving the target argument blank will enable god mode on the calling player.

!hp {target} {health}

Admin only. Gives the target player the specified number of health. Use '^' as the target to give health to yourself, and use '*' as the target to give everyone health.

!freeze [target]

Admin only. Freezes the target player in place so they cant move or use weapons. Leaving the target argument blank will freeze the calling player.
