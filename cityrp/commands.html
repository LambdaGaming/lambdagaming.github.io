Chat Commands

The following is a list of chat commands and their functions for the CityRP server. Angled brackets { } in a command means it's a required argument. Square brackets [ ] in a command means it's an optional argument. If you are viewing this from in-game, a list of chat commands for DarkRP can be found on the left.

!fixeco

Mayor only. Adds 30 eco points at the cost of $300 from the mayor's funds.

!fireoff

Admin only. Extinguishes all of the fires currently burning on the map.

!addons

Available to all players. Opens the page to the addon collection in your Steam overlay.

!group

Available to all players. Opens the page to the Lambda Gaming Steam group in your Steam overlay.

!rules

Available to all players. Opens the CityRP Rules and Guidelines page on this website in your Steam overlay.

!alert

Available to all players. Alerts all police and emergency responders to your location. Requires that you have life alert, which can be purchased at the NPC shop.

!alertcheck

Available to all players. Tells the player whether or not they have life alert equipped.

!showid

Available to all players. Prints in chat the real name, fake name, occupation, and government relationship of the calling player.

!firemayor

IA Agent only. Starts a vote to fire the mayor. 75% of players must vote in favor for it to pass.

!currentevent

Available to all players. Prints the current event in chat. Prints N/A if there is currently no event active.

!noclip [target]

Admin only. Allows players to fly through solid objects. Leaving the target argument blank will enable noclip on the calling player.

!god [target]

Admin only. Prevents the target player from being damaged. Leaving the target argument blank will enable god mode on the calling player.

!hp {target} {health}

Admin only. Gives the target player the specified number of health. Use '^' as the target to give health to yourself, and use '*' as the target to give everyone health.

!freeze [target]

Admin only. Freezes the target player in place so they cant move or use weapons. Leaving the target argument blank will freeze the calling player.

!placehit

Available to all players. Opens the hitman menu where players can request a hit on anyone.

!undercover

Available to all players. Changes the calling players job to Undercover Officer without notifying the rest of the server about the change. It also changes the calling players job name to Citizen to prevent cover from being blown on the scoreboard.

!busstop

Bus driver only. Used during the bus passenger random event when the bus driver is in range of the destination to let the passenger know they can get off.
