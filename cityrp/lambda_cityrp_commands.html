CityRP Chat Commands

The following is a list of chat commands and their functions for the CityRP server. Angled brackets { } in a command means it's a required argument. Square brackets [ ] in a command means it's an optional argument.

!fixeco

Mayor only. Adds 30 eco points at the cost of $3000 from the mayor's funds.

!fireoff

Superadmin only. Extinguishes all of the fires currently burning on the map.

!addons

Available to all players. Opens the page to the addon collection in your Steam overlay.

!group

Available to all players. Opens the page to the Lambda Gaming Steam group in your Steam overlay.

!alert

Available to all players. Alerts all police and emergency responders to your location. Requires that you have life alert, which can be purchased at the NPC shop.

!firemayor

IA Agent only. Fires the mayor for corruption and bans him from that job for 10 minutes. Misuse of this command will result in punishment.

!currentevent

Available to all players. Prints the current event in chat. Prints N/A if there is currently no event active.

!enableproppay

Superadmin only. Allows everyone to spawn props without paying.

!disableproppay

Superadmin only. Restricts prop spawning to the shopkeeper job, who has to pay for each prop spawned.

!noclip [target]

Admin only. Allows players to fly through solid objects. Leaving the target argument blank will enable noclip on the calling player.

!god [target]

Admin only. Prevents the target player from being damaged. Leaving the target argument blank will enable god mode on the calling player.

!hp {target} {health}

Admin only. Gives the target player the specified number of health. Use '^' as the target to give health to yourself, and use '*' as the target to give everyone health.

!freeze [target]

Admin only. Freezes the target player in place so they cant move or use weapons. Leaving the target argument blank will freeze the calling player.

!placehit

Available to all players. Opens the hitman menu where players can request a hit on anyone.
