Status Meanings

  • Active - Project is actively being worked on. Feature updates can come at any time and suggestions are welcome.
  • Inactive - Project is dormant. Updates outside of bug fixes are unlikely, but not impossible. Feature requests are typically not taken.
  • Archived - Project is no longer being worked on at all. No updates will be pushed and no suggestions will be taken. Any GitHub repos associated with the project are archived.
  • Private - Project is currently not public, and may or may not be in active development. If you're interested in the project, you can try asking me for access, but I can't guarantee you'll get it.
