Welcome!

About Us:

We are a small gaming community mainly composed of a group of friends who used to play on a bunch of different Garry's Mod servers and came together to form this community.

We occasionally host our own game servers, some of which contain a lot of custom content created by the owner, OP AKA Source 2 Gman. Most of this custom content is open source and can be found here.

We do not run our servers 24/7. Instead, we open servers based on requests and have play sessions with community members that last a few hours on average.

We are mostly active on our Discord where we talk about games, game modding, programming, etc. This is also where we announce server openings. Come join us!

Our Main Servers:

There are many more games and servers we play on but these are the ones we play on the most.

  • Garry's Mod CityRP - Plays like the default DarkRP gamemode but more in-depth and takes place in a modern city environment. Rules are similar to those seen on Lawless RP servers.

  • Garry's Mod Half-Life Universe RP - This is our custom gamemode based on the major events of the full Half-Life storyline. It is divided up into 3 sections: Black Mesa RP, City 17 RP, and Outland RP. The transition to each section is controlled by the players. More info about this server can be found here.

  • Garry's Mod Sandbox - Simple sandbox server with many addons and maps, and few rules. Addons can be added or removed if enough players request it.

  • Garry's Mod Various Gamemodes - Contains a wide variety of gamemodes and maps that can be switched on the fly if enough players request it. The current list of gamemodes on this server can be found here.

  • Minecraft Vanilla - Always runs the latest release version of the game. Runs multiple maps that can be switched to at the community's request.

  • Minecraft Modded - Normal survival with basic rules. Has around 50 mods with a single map on version 1.12.2. Mods are no longer added or removed on this server to avoid corrupting parts of the map.

  • Minecraft Modded Unbalanced - Normal survival with little rules. Has around 60 mods that were rejected from the other servers. Runs a single map on version 1.12.2. Mods are no longer added or removed on this server to avoid corrupting parts of the map.

  • Minecraft Modded Unbalanced 2 - Same as Unbalanced but with different mods, a new map, and more straight-forward rules. Runs on version 1.12.2.

  • Minecraft Modded Earth - Runs the 1:1 scale Earth mod along with a few other mods. Players are allowed to use creative mode and admin commands to build their bases and explore but if they want to fight another player they are restricted to survival mode without admin commands.

  • Minecraft Modded RLCraft - Runs the most recent version of the RLCraft modpack on a single map.

  • SCP: Secret Laboratory - Frequently updated with different server plugins (including a few custom ones) to keep the game interesting.
