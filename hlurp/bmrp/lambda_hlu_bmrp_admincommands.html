BMRP: Facility Administrator Chat Commands

"!alarm" Remotely activates the facility alarm system.

"/admindamage" Used to call service personnel to a certain sector for damage control.

"/adminevacuate" Used to tell all personnel in sector D to evacuate.

"/adminreport" Used to tell personnel to bring reports to the admin.

"/reportviolation" Used to tell all personnel to look out for any security violations.

"/scientistevacuate" Used to tell all science personnel to evacuate sector C.

"/securityreport" Used to tell security personnel to report to sector D.

"/biohazard" Used to announce that there is a biohazard somewhere in the facility.

"/servicereport" Used to tell service personnel to report somewhere.

"/hallaccess" Used to announce that the maintenance door to Sector C is unlocked.

"/reportams" Used to tell all personnel to report to the AMS.

"/mendown" Used when players are killed.

"/checkelevator" Used to tell service personnel to inspect a broken elevator.

"/tramworking" Used to announce that the trams are running on time.

"/badenergy" Used to announce that there are high energy fields in sector D.

"/detonations" Used to announce that explosions were detected.

"/failelevator" Used to announce to not use the Sector C elevator.

"/aliens" Used to announce alien life was detected in sector C.
