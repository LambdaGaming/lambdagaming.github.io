Black Mesa RP Job: HECU Captain

Note: For a list of commands that go along with this job, click here.

In-Game Description

WARNING! CASCADE EVENT ONLY! USING THIS JOB WITHOUT ADMIN APPROVAL WILL GET YOU DEMOTED BACK TO CIVILIAN! As the HECU Captain, your job is to set up a team of marines to assist Black Mesa security in major hostile events. You can also command your marines in non-hostile events so they can patrol with security or help a fellow marine in training.


Duties:

  • Command and control your marines so they can help security with patroling and emergency situations.

  • Help train new marines and security officers.

  • Talk with the Administrator and Government Man about security concerns if needed.

In the Event of an Emergency:

If a major event happens, such as a cascade, you have to pick what you want to do with your squad. You can either (1)Send out a search and rescue order so your marines save all personnel and bring them topside for questioning or (2)Send out an order to kill all living organisms to try and cover up what happened from the rest of the world.


Special Job-Specific Items:

  • Has ability to command all marines.

  • Has ability to command and control lower-ranking personnel such as scientists and security officers in emergency situations.

  • Has access to military-grade weapons as well as the on-site military base.

Job-Specific Rules/Tips:

  • You only officially have command over your squad of marines, so don't go around the facility thinking you own the place.
