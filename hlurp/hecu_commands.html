BMRP HECU Captain Chat Commands

"/militaryvox" Used to announce that the announcement system is being controlled by the military.

"/militarykos" Used to tell all military personnel to kill all Black Mesa personnel.

"/militarycallsecurity" Used when the military invades to call all security personnel topside for questioning.

"/militarycallscience" Same as above, but for science personnel.

"/militaryforce" Used to tell military personnel to use any force to kill Black Mesa personnel.

"/militarybackup" Used to call for more marines to a certain sector.
